## התקנה מהירה

### דרישות
- Python 3.9+
- Roblox Studio
- OpenAI API Key (חינם למתחילים)
- Anthropic API Key (חינם למתחילים)

### צעדים
```bash
# 1. התקן תלויות
pip install -r requirements.txt

# 2. הגדר API keys
echo "OPENAI_API_KEY=your-key" > .env
echo "ANTHROPIC_API_KEY=your-key" >> .env

# 3. הרץ!
cd src
python main.py
```

📖 **מדריך מפורט**: ראה [INSTALLATION.md](docs/INSTALLATION.md)
